Nếu là đường dẫn file trong repo (vd `resources/...` hoặc `documents/...`) thì đọc file đó trước.

Trả lời theo 5 phần:

### 1. Tổng quan
- Chủ đề là gì, thuộc cert nào, tầm quan trọng (% câu hỏi trong đề)

### 2. Nội dung chính
- Tóm tắt từng mục bằng tiếng Việt
- Giữ nguyên thuật ngữ kỹ thuật tiếng Anh (**in đậm**), giải thích lần đầu xuất hiện
- Code example cho mỗi khái niệm quan trọng
