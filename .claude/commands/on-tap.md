Tạo một buổi ôn tập nhanh cho chủ đề: **$ARGUMENTS**

Trả lời theo đúng 3 phần:

### Phần 1: Tóm Tắt Lý Thuyết (5-10 điểm chính)
- Bullet points ngắn gọn bằng tiếng Việt, kèm code snippet nếu cần
- Nhấn mạnh những điểm dễ nhầm lẫn

### Phần 2: Câu Hỏi Luyện Tập (5-10 câu, format đề thật)
```
Câu X: [Nội dung câu hỏi tiếng Anh]

A. ...   B. ...   C. ...   D. ...

> Đáp án: [X]
> Giải thích: [Tiếng Việt — vì sao đúng, vì sao các đáp án khác sai]
```

### Phần 3: Những Lỗi Thường Gặp
- 3-5 lỗi hay mắc với chủ đề này + cách phân biệt / ghi nhớ

Cuối buổi: gợi ý nếu thấy yếu thì dùng `/lich-su-sai [chủ đề]` để ghi vào sổ lỗi.
