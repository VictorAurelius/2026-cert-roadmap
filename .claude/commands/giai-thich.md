Giải thích khái niệm sau bằng tiếng Việt, dễ hiểu cho người đang ôn thi chứng chỉ: **$ARGUMENTS**

Trả lời theo đúng định dạng:

1. **Định nghĩa ngắn gọn** (1-2 câu tiếng Việt)
2. **Giải thích chi tiết** (tiếng Việt, có ví dụ minh hoạ)
3. **Code ví dụ** (nếu áp dụng được)
4. **Lưu ý thi cử** — những điểm hay xuất hiện trong đề 1Z0-830 / 1Z0-071 / 2V0-72.22, các bẫy/edge case
5. **Từ khoá tiếng Anh liên quan** — giúp tra cứu thêm trong tài liệu gốc

Phạm vi: Java SE 21 (features, OOP, concurrency, modules, records, sealed classes, pattern matching...), Oracle SQL (DDL, DML, DQL, joins, subqueries, functions, indexes, constraints...), Spring (IoC, DI, AOP, Spring Boot, MVC, Data, Security...).

Giữ nguyên thuật ngữ kỹ thuật tiếng Anh, chú thích nghĩa lần đầu xuất hiện.
