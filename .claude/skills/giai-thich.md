# Skill: Giải Thích Khái Niệm ($giai-thich)

Khi người dùng dùng lệnh `/giai-thich [khái niệm]`, hãy thực hiện:

## Mục đích
Giải thích một khái niệm Java / Oracle SQL / Spring bằng tiếng Việt, dễ hiểu cho người học ôn thi chứng chỉ.

## Định dạng đầu ra

1. **Định nghĩa ngắn gọn** (1-2 câu tiếng Việt)
2. **Giải thích chi tiết** (tiếng Việt, có ví dụ minh hoạ)
3. **Code ví dụ** (nếu áp dụng được)
4. **Lưu ý thi cử** — những điểm hay xuất hiện trong đề thi 1Z0-830 / 1Z0-071 / 2V0-72.22
5. **Từ khoá tiếng Anh liên quan** — giúp tra cứu thêm trong tài liệu gốc

## Phạm vi áp dụng
- Java SE 21: features, APIs, OOP, concurrency, modules, records, sealed classes, pattern matching...
- Oracle SQL: DDL, DML, DQL, joins, subqueries, functions, indexes, constraints...
- Spring Framework: IoC, DI, AOP, Spring Boot, Spring MVC, Spring Data, Spring Security...

## Ví dụ sử dụng
```
/giai-thich sealed class
/giai-thich LEFT JOIN vs INNER JOIN
/giai-thich Spring Bean lifecycle
```
