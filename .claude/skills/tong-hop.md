# Skill: Tổng Hợp Tài Liệu ($tong-hop)

Khi người dùng dùng lệnh `/tong-hop [chủ đề hoặc file]`, hãy thực hiện:

## Mục đích
Đọc và tổng hợp tài liệu tiếng Anh (từ file trong repo, URL, hoặc đoạn paste vào) thành bản tóm tắt tiếng Việt có cấu trúc, phù hợp để ôn thi.

## Định dạng đầu ra

### 1. Tổng quan (Overview)
- Chủ đề là gì, thuộc chứng chỉ nào
- Tầm quan trọng trong kỳ thi (% câu hỏi)

### 2. Nội dung chính
- Tóm tắt từng mục bằng tiếng Việt
- Giữ nguyên các thuật ngữ kỹ thuật tiếng Anh (in đậm), kèm giải thích lần đầu xuất hiện
- Có code example cho mỗi khái niệm quan trọng

### 3. Bảng So Sánh (nếu có nhiều khái niệm tương tự)
| Khái niệm | Điểm khác biệt | Khi nào dùng |
|-----------|----------------|--------------|

### 4. Checklist Ôn Tập
- [ ] Các điểm PHẢI nhớ trước khi thi
- [ ] Gotchas / edge cases hay bị hỏi

### 5. Câu Hỏi Ôn Tập Nhanh (3-5 câu)
Câu hỏi ngắn dạng fill-in hoặc true/false để tự kiểm tra nhanh.

## Ví dụ sử dụng
```
/tong-hop documents/java/streams.md
/tong-hop "sealed classes and interfaces in Java 21"
/tong-hop Spring Boot Auto-Configuration
```
