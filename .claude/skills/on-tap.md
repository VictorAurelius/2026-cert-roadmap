# Skill: Ôn Tập Chủ Đề ($on-tap)

Khi người dùng dùng lệnh `/on-tap [chủ đề] [chứng chỉ?]`, hãy thực hiện:

## Mục đích
Tạo buổi ôn tập nhanh một chủ đề, bao gồm tóm tắt lý thuyết + câu hỏi kiểm tra theo format đề thi thật.

## Định dạng đầu ra

### Phần 1: Tóm Tắt Lý Thuyết (5-10 điểm chính)
- Bullet points ngắn gọn bằng tiếng Việt
- Kèm code snippet nếu cần
- Nhấn mạnh những điểm dễ nhầm lẫn

### Phần 2: Câu Hỏi Luyện Tập (5-10 câu)
Format giống đề thi thật:
```
Câu X: [Nội dung câu hỏi tiếng Anh]

A. [Lựa chọn A]
B. [Lựa chọn B]
C. [Lựa chọn C]
D. [Lựa chọn D]

> Đáp án: [X]
> Giải thích: [Tiếng Việt, giải thích tại sao đáp án đúng và tại sao các đáp án khác sai]
```

### Phần 3: Những Lỗi Thường Gặp
- Liệt kê 3-5 lỗi hay mắc phải với chủ đề này
- Cách phân biệt / ghi nhớ

## Chứng chỉ hỗ trợ
- `1Z0-830` hoặc `java`: Java SE 21 Developer
- `1Z0-071` hoặc `sql`: Oracle Database SQL Associate
- `2V0-72.22` hoặc `spring`: Spring Certified Professional

## Ví dụ sử dụng
```
/on-tap records java
/on-tap GROUP BY sql
/on-tap AOP spring
/on-tap lambda expressions
```
