# documents/output/ — Bản HTML người đọc

Thư mục chứa **sản phẩm HTML** render từ tài liệu MD nguồn (bằng `html-anything`).
Đây là bản để **học / ôn / in / chia sẻ** — mở trực tiếp bằng browser.

- **Nguồn (sửa ở đây):** `documents/*.md`, `.claude/plans/*.md`
- **Render:** `scripts/render-html.sh <file.md> [template]` → ra file `.html` ở đây
- **Workflow + map template:** [`.claude/plans/html-output-workflow.md`](../../.claude/plans/html-output-workflow.md)

> File `.html` ở đây được commit (là deliverable). Đừng sửa tay — sửa MD nguồn rồi render lại.
